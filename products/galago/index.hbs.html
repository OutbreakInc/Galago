What is Galago?

Galago is a rapid electronics prototyping board that combines a powerful ARM Cortex-M3 microcontroller with a hardware debugger in a tiny format, allowing hobbyists and professionals alike to turn project ideas into reality faster and better than other microcontroller platforms.

Use it!

Plug Galago into a standard breadboard for quick prototyping, pop it into an app board to build an application or solder it directly into your commercial product.

Developing with Galago

Galago plugs into a standard solderless breadboard or an app board and connects to your computer with USB. The USB connection permits downloading and debugging firmware, plus it will power Galago if it's not connected to another power source.

Write easy-to-understand code - built on C++ with Galago's simple but powerful development tool and deploy it to the hardware with a single click.

Use the integrated debugger to pause code execution, inspect variables and continue running - this helps you quickly find software problems as they arise to fix them faster with less head-scratching.

Debugger

Software usually doesn't work the first time, even for experienced engineers.

A debugger is a tool that lets you stop a program as it's running so you can inspect the values of variables and memory. This lets you determine where and why it went wrong. You may also step through the program as Galago runs each line of your code so you can see how it flows. At your command, the program will continue running or start over.

A debugger gives you the information you need to fix bugs in your code. Galago's debugger, built into the development tool, makes it easy to fix your code and download the revised program with just another click.

Technical specifications

Board features

  • 72 MHz 32-bit ARM CPU with 32KB of flash ROM and 8KB of RAM
  • Integrated hardware debugger
  • One high-speed SPI port, up to 36 Mbps
  • One high-speed I2C port, up to 1.5 Mbps
  • One UART/USART with hardware flow-control capability, up to 256 kbps
  • 10 high-speed PWM pins, 6 driven by 32-bit (high-resolution) timers
  • 6 ADC (analog) input pins with 10-bit resolution at over 400 KHz
  • 25 GPIO (digital) input/output pins including the on-board LED
    • (Many features share pins, therefore not all can be used at once.)

Dimensions

Galago measures 0.76 x 1.46 in., 19.2 x 37.0 mm, with pins positions identical to 0.6" 28-DIP.
