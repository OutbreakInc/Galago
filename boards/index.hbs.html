App Boards

Accelerate your projects

App Boards for Galago get you to your goal faster by building popular feature packages into a single board.

Current App Boards

  • Blueblock: Bluetooth®
  • Etherblock: Ethernet
  • Explorer Board: GPS, Accelerometer
  • LEDBlock: 16-channel PWM
  • Audioblock: Stereo DAC, Amplifier
  • Shield Adapter (for Arduino® shields)

Blueblock Bluetooth® App Board

Cut the cable with wireless serial port feature of Blueblock. When enabled, Galago's io.serial object is connected to the board's Bluetooth® module.

Blueblock also features convenient breadboard-style prototyping holes on both sides of the board.

Learn more »

Etherblock Ethernet App Board

Connect your Galago to a wired network with Etherblock. Also features an on-board switch-mode power supply and a microSD socket.

Etherblock implements the Microchip ENC28J60 Ethernet controller chip.

Learn more »

Explorer GPS App Board

Reach further, higher and faster with the Explorer board. Featuring a miniature GPS module with a tiny, surface-mount antenna and a three-axis accelerometer, the Explorer board is an ideal starting point for tracking, guidance and exploration projects.

It's also easy to store logs or load geographic data from the Explorer board's microSD slot.

Learn more »

Technology Accessory

Logiblock App Boards for Galago have auto-detection and configuration features built in. Galago can detect which App Board(s) it is connected to and load library features automatically.

Unlike Arduino® shields, App Boards will not electrically conflict or short each other out if stacked.

LEDBlock LED and PWM App Board

Control up to 16 LED or servo channels with 12-bit (4096-level) resolution with the LEDBlock. If used for LEDs, constant-current-sunk chains up to 10 LEDs long - 17 Volts - are possible.

Based on the TI TLC5940 chip, meaning multiple chips can be chained for an almost unlimited number of channels.

Learn more »

Audioblock Stereo DAC App Board

Synthesize or play back recorded audio with Audioblock. Features two 1.5W class-D audio amplifiers, a 16-bit 48KHz stereo DAC and a microSD socket for storage. Also sports a low-noise switch-mode power supply.

based on the NXP UDA1334ATS digital-to-analog converter chip.

Learn more »

Shield Adapter for Arduino® shields

Adapts Galago to connect to Arduino® shields. SPI, I2C, UART and ADC pins maintain a logical mapping to their shield counterparts.

No specific software libraries are provided for Arduino® shields, but Galago API makes interfacing with hardware accessories easy.
